# RcAct_RcCar
SSAFY 12기 RcCar pose control project


```markdown
## 설치 방법

1. **가상환경 설정 및 활성화**
   ```bash
   python3 -m venv myenv
   source myenv/bin/activate
   ```

2. **필수 패키지 설치**
   ```bash
   pip install -r requirements.txt
   ```

`requirements.txt` 파일에는 이 프로젝트에 필요한 모든 의존성이 포함되어 있습니다.
```
